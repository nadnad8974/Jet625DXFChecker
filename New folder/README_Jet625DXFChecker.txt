DONATONI JET 625 DXF CHECKER

FASTEST WAY TO RUN IT
1. Put all files in one folder.
2. Double-click: Run_Jet625DXFChecker.bat
3. It will install ezdxf automatically and open the app.

IF YOU WANT A REAL EXE
1. Double-click: Build_EXE_Jet625DXFChecker.bat
2. Wait for it to finish.
3. Your EXE should be in:
   dist\Jet625DXFChecker.exe

WHAT IT CHECKS
- required layer exists
- top layer exists
- closed POLYLINE / LWPOLYLINE
- warns about loose LINE / ARC / SPLINE entities
- can save a fixed DXF copy

DEFAULT LAYER NAMES
- Required polyline layer: CUT
- Top layer: TOP

YOU CAN CHANGE THE LAYER NAMES INSIDE THE APP.

IMPORTANT
If your Donatoni needs exact layer names other than CUT and TOP, tell ChatGPT the exact names and it can hard-code them for you.
